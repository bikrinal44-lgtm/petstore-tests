# petstore-tests
Автоматизированные тесты для Swagger Petstore API(https://petstore.swagger.io)

# Стек
- Python3
- pytest(https://github.com/pytest-dev/pytest)
- requests(https://github.com/psf/requests)

# Установка и запуск
1. Клонируйте репозиторий
2. Откроте у себя в системе
3. Установите зависимости: pip install -r requirements.txt
4. Запуститке тест: pytest
